Store event photos for the events page here.

Recommended pattern:

```ts
import networkingSocialImage from '@/assets/events/networking-social.jpg'
```

Then assign the imported file to the `image` field in `src/components/Events.tsx`.

Use short, predictable file names such as:

- `networking-social.jpg`
- `brainteaser-battle.jpg`
- `semester-kickoff.jpg`
