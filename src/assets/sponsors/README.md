Store sponsor logos here.

Matching rule:
- The filename is matched to the sponsor name after lowercasing and removing spaces/punctuation.

Examples:
- `airwallex.svg`
- `jane-street.png`
- `australiansuper.webp`
- `macquarie-logo.svg` only if you set `logo: 'macquarie-logo'` in `src/components/Sponsors.tsx`
